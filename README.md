Task 1: Factorial Calculator
How It Works
1. A function is defined to compute factorial using a `for` loop.
2. The user enters a number.
3. The program multiplies numbers from `1` to `n` to compute the factorial.
4. The result is displayed.
Steps to Run
1. Open your terminal or Python IDE.
2. Navigate to the folder where the script is stored.
3. Run the program:
   ```bash
   python task1.py
   ```
Task 2: Using the Math Module for Calculations
How It Works
1. The user enters a number.
2. The program uses:
  - math.sqrt() for square root
  - math.log() for natural logarithm
  - math.sin() for sine value
3. The results are displayed to the user.
Steps to Run
1. Open your terminal or Python IDE.
2. Navigate to the folder where the script is stored.
3. Run the program:
   ```bash
   python task2.py
   ```
